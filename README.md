# Eddy's Portfolio

The story of my life. And beyond.

## Tech Stack

This portfolio is built with a modern stack focusing on performance, 3D interactivity, and clean design:

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Core:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **CMS:** [Hygraph](https://hygraph.com/) (GraphQL)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics:** [Three.js](https://threejs.org/) & [React Three Fiber](https://docs.pmnd.rs/react-three-fiber)
- **Forms:** [Web3Forms](https://web3forms.com/)
- **Icons:** [Phosphor Icons](https://phosphoricons.com/)
- **Security:** [hCaptcha](https://www.hcaptcha.com/)
